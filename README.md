Классическая игра «Змейка» на Python с использованием Tkinter Canvas. Реализован зацикленный мир (телепортация через границы), ускорение с ростом счёта и динамическая генерация яблок.

Управление: Стрелки (↑ ↓ ← →).

Особенности:

Чистый код (≈125 строк).

Отрисовка через Canvas с полной перерисовкой кадра.

Защита от разворота в себя и столкновения с телом.
